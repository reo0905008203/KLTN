# Website WEBSITE BÁN MÁY TÍNH TÍCH HỢP VỚI CHATBOX bằng Nodejs

Một ứng dụng web giúp quản lý đơn hàng và sản phẩm cho cửa hàng PC và linh kiện máy tính, được xây dựng tương tự như dự án TTTN.

## Tính năng chính

- ✅ Quản lý sản phẩm
- ✅ Giỏ hàng và đặt hàng online
- ✅ Quản lý đơn hàng
- ✅ Đăng nhập và phân quyền người dùng
- ✅ Tìm kiếm sản phẩm
- ✅ Lịch sử đơn hàng
- ✅ Chatbox

## Cài đặt môi trường
Tiến hành tải nodeJS
Và import csdl từ file PCShop.sql

## Cấu trúc thư mục
- `controllers/` - Chứa logic xử lý nghiệp vụ 
- `models/` - Các tác vụ thao tác trên cơ sở dữ liệu
- `routes/` - Định tuyến API
- `views/` - Giao diện người dùng (EJS)
- `public/` - Static files như CSS, JS, hình ảnh

## Công nghệ sử dụng
- Node.js + Express.js
- SQL Server
- EJS (giao diện)
